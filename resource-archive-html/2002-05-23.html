ORAL ANSWER TO QUESTION
MOBILE PHONE USERS
(Mobile phone portability for)


    1. Mr Tan Soo Khoon asked the Acting Minister for Information, Communications and the Arts if the Infocomm Development Authority will allow full mobile phone portability for mobile phone users so that they can retain their existing telephone numbers when they switch from one service provider to another.

    The Acting Minister for Information, Communications and the Arts (Mr David T. E. Lim): Sir, mobile number portability refers to the ability of mobile customers to retain their telephone number when they switch between the different mobile operators. IDA mandated mobile number portability since the start of mobile competition on 1st April 1997, when M1 entered the market.

    Several technical solutions can be used to achieve mobile number portability. In line with IDA's principle of technology neutrality, IDA did not mandate which method operators should use. Up until now, the implementation of mobile number portability has been industry-driven.

    IDA recognises that the current mobile number portability solution adopted by the mobile operators has a number of limitations. Specifically, SMS messages are not portable. This means that a user cannot receive SMS messages sent to his original number once he switches to a different mobile operator. Given the high usage of SMS in Singapore, and the projected proliferation of mobile data services, the lack of SMS portability could become a significant barrier to migration between mobile operators. This would then inhibit mobile competition.

    The limitation on SMS portability can be overcome. However, this requires additional investments by the mobile operators, and the end users could end up bearing the higher cost.

    In view of the potential impact on both the mobile operators and customers, IDA has decided to re-convene the Mobile Number Portability (MNP) Working Group so that the mobile operators could engage in a discussion of all the implications of a migration to a full mobile number portability solution. This would allow IDA to weigh the pros and cons, including the likely impact on consumers, before deciding whether or not it is timely to mandate full number portability.

But if someone calls his old telephone number, he will still receive the call. So, in that sense, his number has not changed to the people who are calling him, and that makes it portable.
